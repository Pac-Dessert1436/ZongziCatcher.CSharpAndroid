# Zongzi Catcher (C#, MonoGame Android) 🛶🍙

A Dragon Boat Festival-themed arcade game for Android, ported to C# and .NET 10.0+.

The original F# implementation of this game can be found [here](https://github.com/Pac-Dessert1436/ZongziCatcher.FSharpGame).

## 🎮 How to Play

**Controls:**
- Tap arrow buttons (letterbox areas) to move left/right
- Tap game area to pause
- Tap anywhere to start/resume/restart

**Rules:**
- Catch zongzi: +10 points
- Avoid scorpions: -1 life
- Miss zongzi: -5 points
- 3 lives to survive!

## 🛠️ Requirements
- .NET 10.0+
- MonoGame SDK
- Android SDK

## 📦 Clone Repository
```bash
git clone https://github.com/Pac-Dessert1436/ZongziCatcher.CSharpAndroid.git
```

## 📦 Build
```bash
cd ZongziCatcher.CSharpAndroid
dotnet restore
dotnet build
```
